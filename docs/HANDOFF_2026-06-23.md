## 体験フロー

```
[Introシーン]
  mainVisual.png 表示
  → M5接続待ち（AWAITING DEVICE...点滅）
  → 接続後「PRESS ANY BUTTON TO START」
  → BtnB or BtnC → フェードアウト → Mainシーンへ

[Mainシーン / FlowState]
  Booting
    → M5から BOOT_DONE 受信 or 12秒タイムアウト

  Login (Plane0: loginUI.png)
    → BtnB → 全画面グリッチ + hackPopupUI表示

  HackReady (Plane0)
    → BtnB → CMD,HACK_START 送信 → ハックミニゲーム開始

  Hacking (Plane0)
    → 成功 → LogDisplay へ
    → 失敗 → hackPopupUI再表示 → HackReady へ（リトライ可）

  LogDisplay (Plane1: logDisplay.png)
    → BtnC → DownloadReady へ

  DownloadReady (Plane2: downloadWindow.png)
    → BtnB → ダウンロード演出開始

  Downloading
    → downloadDuration秒後 → DownloadDone

  DownloadDone
    → TRANSFER COMPLETE 表示（終了）
```

## 通信プロトコル

M5 → Unity:
```
M5_HANDSHAKE_v1          （起動後1秒間、100ms間隔で送信）
D,Pitch,Roll,AX,AY,AZ,TX,TY,BtnA,BtnB,BtnC,HackState,HackProgress
BOOT_DONE                （起動完了時に1回）
```

Unity → M5:
```
CMD,CONNECTED            （ハンドシェイク成功後に1回）
CMD,HACK_START           （ハック開始時）
```

HackState: `0=IDLE 1=INIT 2=T1 3=T2 4=T3 5=TL1 6=TL2 7=TL3 8=SUCCESS 9=FAIL`

| hack_fail.wav | ハック失敗時 | 同上 |

ch0: WAV再生 / ch1: UIトーン / ch2: Tiltホールドフィードバック

## PostEffectシェーダー（PostEffect.shader）

スクリーンスペースで以下をこの順に処理:

1. **スクリーングリッチ** (`_GlitchIntensity`, `_GlitchSeed`)
   - 太いブロックずれ（4%高さ単位）
   - 細いラインティア（0.5%単位）
   - RGB チャンネル個別ずれ
2. **CRT ラスターウォブル** (`_WobbleIntensity`) — スキャンライン波打ち
3. **色収差** (`_AberrationStrength`)
4. **スキャンライン** (`_ScanlineCount`, `_ScanlineIntensity`)
5. **フィルムグレイン** (`_GrainStrength`)
6. **ゴーストグロー** (`_GhostOffset`)
7. **ビネット**
8. **モノクロ** (`_MonoIntensity`) — リン光グリーン変換

## M5VisualController グリッチ設定

| パラメータ | デフォルト | 意味 |
|---|---|---|
| Glitch Interval Min | 2 秒 | 自動グリッチの最小間隔 |
| Glitch Interval Max | 7 秒 | 自動グリッチの最大間隔 |
| Glitch Duration | 0.8 秒 | 1回のグリッチ持続時間 |
| Glitch Intensity | 0.65 | グリッチ強度 |
| Mono Glitch Chance | 0.35 | グリッチ時にモノクロになる確率 |

## 再開時の手順

1. **Unity を起動**してプロジェクトを開く
2. M5 を USB-C 接続
3. **M5ファームウェア書き込み**（まだ済んでいない場合）
   - Arduino IDE で書き込み → `Connecting......` でRSTボタン
4. Unity Inspector で `Glitch Interval Min/Max` を 2 / 7 に確認・設定
5. Introシーンから Play で動作確認
